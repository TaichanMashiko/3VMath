📖 分数関数のグラフと漸近線

分数関数 $y = \frac{3}{x-1}$ について考えます。

漸近線の確認

この関数の漸近線は以下の通りです。 ★★☆

  • $x = 1$
  • $y = 0$

$y$ の漸近線は、分数式の外側に加えられる数によって決まります。この関数 $y = \frac{3}{x-1}$ は、$\frac{3}{x-1} + 0$ と見なせるため、$y$ の漸近線は $y=0$ となります。もし関数が例えば $y = \frac{3}{x-1} + 2$ であれば、$y$ の漸近線は $y=2$ となります。

元となるグラフの確認

この関数 $y = \frac{3}{x-1}$ の元となるグラフは $y = \frac{3}{x}$ です。 ★★☆ この $y = \frac{3}{x}$ のグラフを $x$軸方向に $1$ だけ平行移動したものが $y = \frac{3}{x-1}$ のグラフとなります。

✍️ 例題1: 分数関数のグラフ描画 ★☆☆ ★★☆

関数 $y = \frac{3}{x-1}$ のグラフを描画します。

グラフの描画手順

  1. まず、x軸とy軸を準備します。
  2. 次に、漸近線 $x=1$ と $y=0$ (x軸) を描きます。見やすくするために、漸近線に色を付けても良いでしょう。
  3. 元となるグラフ $y = \frac{3}{x}$ は、分子の $3$ が正なので、グラフは右上と左下の領域に描かれる双曲線となります。
  4. $y = \frac{3}{x-1}$ のグラフは、この双曲線を漸近線に触れないように描きます。

➕ グラフの概略図（イメージ）

（実際の授業では黒板に描画されますが、ここでは説明に基づいたイメージです）

x=1 と y=0 (x軸) を漸近線として、第1象限と第3象限に対応する位置に双曲線が描かれます（ただし座標軸の原点ではなく、漸近線の交点(1,0)を基準とした象限）。

y軸との交点

グラフがy軸と交わる点の座標を求めます。y軸との交点では $x=0$ ですから、関数に $x=0$ を代入します。

$y = \frac{3}{0-1} = \frac{3}{-1} = -3$

したがって、y軸との交点は $(0, -3)$ です。グラフを描く際にこの点を通るようにします。

✍️ 例題2: 分数関数と直線の交点 ★★☆ ★★★

問題設定

関数 $y = \frac{3}{x-1}$ と直線 $y = x+1$ との共有点のx座標を求めます。

解法: 連立方程式

共有点の座標を求めるには、2つの関数の式を連立させて解きます。

$\frac{3}{x-1} = x+1$

計算過程

両辺に $(x-1)$ を掛けます（ただし $x \neq 1$）。

$3 = (x-1)(x+1)$

$3 = x^2 - 1$

$x^2 = 3 + 1$

$x^2 = 4$

$x = \pm 2$

よって、共有点のx座標は $x=2$ と $x=-2$ です。

グラフ上での確認

直線 $y = x+1$ は、傾きが $1$、y切片が $1$ の直線です。この直線を先ほどの分数関数のグラフに重ねて描くと、2点で交わることが視覚的に確認できます。その交点のx座標が $2$ と $-2$ に対応します。

✍️ 例題3: 不等式の解法 (グラフ利用) ★★☆ ★★☆

問題設定

不等式 $\frac{3}{x-1} > x+1$ をグラフを用いて解きます。

グラフを用いた考察

この不等式は、関数 $y = \frac{3}{x-1}$ のグラフが、直線 $y = x+1$ のグラフよりも上方にある $x$ の範囲を求めることを意味します。 💡

例題2で求めた共有点のx座標 ($x=-2, x=2$) と、分数関数の漸近線 ($x=1$) が重要な手がかりとなります。

グラフ上で $y = \frac{3}{x-1}$ が $y = x+1$ より上にある部分を探します。

  • $x < -2$ の範囲
  • $1 < x < 2$ の範囲

解の導出と注意点

上記の考察から、不等式の解は次のようになります。

$x < -2$ または $1 < x < 2$

⚠️ 定義域の確認

分数関数 $y = \frac{3}{x-1}$ は、$x=1$ で定義されません（分母が0になるため）。したがって、解の範囲を考える際には、この点を除外する必要があります。今回の場合、$1 < x < 2$ という解の区間は $x=1$ を含んでいないため問題ありません。

💡 Tips/補足説明

  • 漸近線の重要性: 分数関数のグラフを描く上で、漸近線を正確に把握することが最も重要です。漸近線はグラフの「骨格」となります。
  • 対称性: 分数関数 $y = \frac{k}{x-p} + q$ のグラフは、点 $(p, q)$（漸近線の交点）に関して対称です。この性質を知っていると、グラフの形状をより正確にイメージできます。
  • グラフの活用: 不等式の問題では、代数的な計算だけでなく、グラフの上下関係を利用することで視覚的に解の範囲を捉えやすくなります。特に複雑な不等式では有効な手段です。

⚠️ 間違いやすいポイント

  • 漸近線の見落とし・誤り: $y = \frac{k}{x-p} + q$ の形のとき、漸近線は $x=p, y=q$ です。特に $q=0$ の場合、$y=0$ (x軸) が漸近線であることを見落とさないようにしましょう。
  • 分子の符号とグラフの向き: $y = \frac{k}{x}$ の形で $k>0$ なら右上と左下 (第1,3象限風)、$k<0$ なら左上と右下 (第2,4象限風) に双曲線が描かれます。平行移動後もこの関係は維持されます。
  • 連立方程式の計算ミス: 分母を払う際に、掛け忘れや符号のミスが起こりやすいです。丁寧な計算を心がけましょう。
  • 不等式の解の範囲: グラフを利用する場合、どの範囲でどちらのグラフが上 (または下) にあるかを正確に読み取ることが重要です。漸近線や交点が範囲の区切り目になります。また、定義域 ($x \neq 1$) にも注意が必要です。

❓ 理解度チェッククイズ

  1. 分数関数 $y = \frac{2}{x+3} - 1$ の漸近線の方程式を答えなさい。

    解答と解説

    解答: $x = -3$, $y = -1$

    解説: $y = \frac{k}{x-p} + q$ の漸近線は $x=p, y=q$ です。この関数は $y = \frac{2}{x-(-3)} + (-1)$ と変形できるので、$p=-3, q=-1$ となります。

  2. 関数 $y = \frac{1}{x}$ のグラフを、x軸方向に $2$、y軸方向に $-3$ だけ平行移動したグラフを表す式を答えなさい。

    解答と解説

    解答: $y = \frac{1}{x-2} - 3$

    解説: x軸方向に $p$、y軸方向に $q$ だけ平行移動する場合、$x$ を $x-p$ に、$y$ を $y-q$ に置き換えます。よって、$y-(-3) = \frac{1}{x-2}$ となり、整理すると $y = \frac{1}{x-2} - 3$ です。

  3. 分数関数 $y = \frac{1}{x}$ と直線 $y = x$ の共有点のx座標をすべて答えなさい。

    解答と解説

    解答: $x = 1, x = -1$

    解説: 連立方程式 $\frac{1}{x} = x$ を解きます。$1 = x^2$ より、$x^2 = 1$。したがって $x = \pm 1$ となります ($x \neq 0$ を満たします)。

✏️ 標準的な練習問題

  1. 関数 $y = \frac{-2}{x+1} + 3$ のグラフを描き、漸近線の方程式とy軸との交点の座標を求めなさい。 ★☆☆

    解答と解説

    漸近線: $x = -1$, $y = 3$

    y軸との交点: $x=0$ を代入すると $y = \frac{-2}{0+1} + 3 = -2 + 3 = 1$。よって $(0, 1)$。

    グラフの概略: 漸近線 $x=-1, y=3$ を描きます。分子が $-2 < 0$ なので、グラフは漸近線の交点を基準として左上と右下の領域に描かれます。点 $(0,1)$ を通ります。

  2. 分数関数 $y = \frac{4}{x-2}$ と直線 $y = -x+6$ の共有点の座標を求めなさい。 ★★☆

    解答と解説

    連立方程式 $\frac{4}{x-2} = -x+6$ を解きます ($x \neq 2$）。

    $4 = (x-2)(-x+6)$

    $4 = -x^2 + 6x + 2x - 12$

    $4 = -x^2 + 8x - 12$

    $x^2 - 8x + 16 = 0$

    $(x-4)^2 = 0$

    $x = 4$ (重解)

    $x=4$ のとき、$y = -4+6 = 2$。

    共有点の座標: $(4, 2)$ (接します)

  3. 不等式 $\frac{1}{x+1} \leq x-1$ を解きなさい。 ★★☆

    解答と解説

    グラフを利用するか、場合分けして代数的に解きます。ここではグラフのイメージで考えます。

    $y = \frac{1}{x+1}$ (漸近線 $x=-1, y=0$) と $y = x-1$ (傾き1, y切片-1) のグラフの上下関係を調べます。

    共有点を求めると: $\frac{1}{x+1} = x-1 \Rightarrow 1 = (x+1)(x-1) \Rightarrow 1 = x^2-1 \Rightarrow x^2=2 \Rightarrow x = \pm\sqrt{2}$。

    グラフを描いて考察すると、分数関数が直線以下になるのは、

    $- \sqrt{2} \leq x < -1$ または $x \geq \sqrt{2}$

    （$x=-1$ は定義域に含まれないため、$<$ となります）

✏️ 発展的な練習問題

  1. 関数 $y = \frac{x+1}{x-1}$ のグラフと直線 $y = mx+2$ が異なる2点で交わるような、定数 $m$ の値の範囲を求めなさい。 ★★★

    解答と解説のヒント

    まず、$y = \frac{x+1}{x-1} = \frac{(x-1)+2}{x-1} = 1 + \frac{2}{x-1}$ と変形します。漸近線は $x=1, y=1$ です。

    次に、$\frac{2}{x-1} + 1 = mx+2$ を $x$ についての2次方程式に変形し、その判別式 $D$ を考えます。

    $\frac{2}{x-1} = mx+1$

    $2 = (mx+1)(x-1) \quad (x \neq 1)$

    $2 = mx^2 - mx + x - 1$

    $mx^2 + (1-m)x - 3 = 0$

    この2次方程式が $x=1$ 以外の異なる2つの実数解を持てばよいことになります。

    1. $m \neq 0$ の場合: 判別式 $D = (1-m)^2 - 4m(-3) = (1-m)^2 + 12m = 1 - 2m + m^2 + 12m = m^2 + 10m + 1 > 0$ を解きます。解の公式より $m = \frac{-10 \pm \sqrt{100-4}}{2} = -5 \pm \sqrt{24} = -5 \pm 2\sqrt{6}$。よって $m < -5-2\sqrt{6}$ または $m > -5+2\sqrt{6}$。 さらに、$x=1$ が解にならない条件を確認します。$m(1)^2 + (1-m)(1) - 3 \neq 0 \Rightarrow m + 1 - m - 3 \neq 0 \Rightarrow -2 \neq 0$ (これは常に成り立つ)。
    2. $m = 0$ の場合: $(1-0)x - 3 = 0 \Rightarrow x=3$ となり、解は1つなので不適。

    したがって、求める $m$ の範囲は $m < -5-2\sqrt{6}$ または $m > -5+2\sqrt{6}$ です。

🔖 授業全体の要約

本日の授業では、分数関数 $y = \frac{k}{x-p} + q$ のグラフの描き方、特に漸近線の重要性について学びました。また、分数関数と直線の共有点を求める問題（連立方程式の解法）や、グラフを利用した不等式の解法についても扱いました。これらの知識は、より複雑な関数の問題を解く上での基礎となります。

  • 分数関数のグラフは漸近線を基準に描く。
  • 分子 $k$ の符号によってグラフの配置が決まる。
  • 分数関数と直線の共有点は、式を連立させて解く。
  • 不等式は、グラフの上下関係を視覚的に捉えることで解の範囲を特定できる。
  • 定義域（分母が0にならない）の確認を忘れない。

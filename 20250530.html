2. 逆関数の基本的な考え方 💡 ★★☆

関数をトンネルとして捉える

関数 \(y = f(x)\) とは、ある値 \(x\) を入力すると、特定の処理（トンネル）を経て別の値 \(y\) が出力される仕組みと考えることができます。

例えば、\(f(x) = x^2\) という関数（トンネル）があるとします。このトンネルに「2」という数字を入れると、「4」という数字が出てきます。また、\(f(x) = x+5\) という関数（トンネル）に「2」を入れると、「7」が出てきます。

逆の対応関係

逆関数とは、この「トンネル」の処理を逆に行うものです。つまり、出力された \(y\) から元の \(x\) を求める対応関係を考えます。もし \(y = f(x)\) という関数があるなら、それに対応する逆の関数が存在するはずです。

3. 逆関数の定義 📖 ★★★

関数 \(y = f(x)\) において、その定義域内の各値 \(x\) に対して、値域内の値 \(y\) がただ一つ定まるときを考えます。このとき、\(y\) の各値に対して、元の \(x\) をただ一つ定める対応関係が存在する場合、この対応関係を \(f\) の逆関数といい、\(y = f^{-1}(x)\) や \(x = f(y)\) のように表します。

記号 \(f^{-1}\) は「エフ インバース」と読みます。これは \(f\) のマイナス1乗という意味ではありません。

⭐ 重要: 逆関数を求める基本的な手順は、元の関数 \(y = f(x)\) の式を \(x\) について解き、最後に \(x\) と \(y\) を入れ替えることです。

4. 逆関数を求める例 ✍️

例1: 一次関数 \(y = 2x + 1\) ★☆☆

関数 \(y = 2x + 1\) の逆関数を求めなさい。

解答と解説

1. まず、与えられた式 \(y = 2x + 1\) を \(x\) について解きます。
\(y - 1 = 2x\)
\(x = \frac{y-1}{2}\)

2. 次に、\(x\) と \(y\) を入れ替えます。
\(y = \frac{x-1}{2}\)

⭐ 答え: \(y = \frac{1}{2}x - \frac{1}{2}\)

例2: 無理関数 \(y = \sqrt{x-1}\) ★★☆

関数 \(y = \sqrt{x-1}\) の逆関数を求めなさい。また、その定義域と値域も示しなさい。

解答と解説

元の関数の定義域と値域:
ルートの中身は0以上である必要があるため、\(x-1 \geq 0\)。よって、定義域は \(x \geq 1\) です。
また、\(\sqrt{\cdot}\) の値は常に0以上なので、値域は \(y \geq 0\) です。

1. \(y = \sqrt{x-1}\) を \(x\) について解きます。
両辺を2乗します（ただし、\(y \geq 0\) であることに注意）。
\(y^2 = x - 1\)
\(x = y^2 + 1\)

2. \(x\) と \(y\) を入れ替えます。
\(y = x^2 + 1\)

逆関数の定義域と値域:
逆関数では、元の関数の定義域と値域が入れ替わります。
したがって、逆関数の定義域は元の関数の値域である \(x \geq 0\) となります。
逆関数の値域は元の関数の定義域である \(y \geq 1\) となります。

⭐ 答え: 逆関数は \(y = x^2 + 1\)。定義域は \(x \geq 0\)、値域は \(y \geq 1\)。

例3: 指数関数 \(y = 2^x\) ★★☆

関数 \(y = 2^x\) の逆関数を求めなさい。

解答と解説

1. \(y = 2^x\) を \(x\) について解きます。これには対数の定義を用います。
\(x = \log_2 y\)

2. \(x\) と \(y\) を入れ替えます。
\(y = \log_2 x\)

元の関数の定義域と値域:
\(y = 2^x\) の定義域は実数全体、値域は \(y > 0\) です。

逆関数の定義域と値域:
逆関数 \(y = \log_2 x\) の定義域は \(x > 0\)（元の関数の値域）、値域は実数全体（元の関数の定義域）となります。これは対数関数の真数条件（真数は正）とも一致します。

⭐ 答え: \(y = \log_2 x\)

5. 元の関数と逆関数のグラフの関係 ⭐ ★★★

関数 \(y=f(x)\) のグラフとその逆関数 \(y=f^{-1}(x)\) のグラフは、直線 \(y=x\) に関して対称になります。

これは、逆関数を求める際に \(x\) と \(y\) を入れ替える操作が、グラフ上では点 \((a, b)\) を点 \((b, a)\) に移すことに対応し、これが直線 \(y=x\) に関する対称移動となるためです。

図: \(y=2^x\)（青線）とその逆関数 \(y=\log_2 x\)（赤線）のグラフは、直線 \(y=x\)（緑破線）に関して対称です。

6. 定義域と値域に関する重要ポイント ⚠️ ★★★

逆関数を考える上で、定義域と値域は非常に重要です。

  • 元の関数 \(y=f(x)\) の定義域は、逆関数 \(y=f^{-1}(x)\) の値域になります。
  • 元の関数 \(y=f(x)\) の値域は、逆関数 \(y=f^{-1}(x)\) の定義域になります。

特に、逆関数を求めた後に、その定義域を正しく設定しないと、元の関数との対応関係が崩れてしまうことがあります。例えば、\(y = x^2\) の逆関数を単純に \(y = \sqrt{x}\) とすると、\(y = x^2\) の定義域が実数全体であるのに対し、\(y = \sqrt{x}\) の値域は \(y \geq 0\) となり、対応がつきません。この場合、元の関数 \(y=x^2\) の定義域を \(x \geq 0\) に制限すれば、値域も \(y \geq 0\) となり、逆関数 \(y=\sqrt{x}\)（定義域 \(x \geq 0\)、値域 \(y \geq 0\)）と1対1に対応します。

7. 練習問題 ✏️ ★★☆

問題1: 問の6番 ★★☆

関数 \(y = \sqrt{x+3}\) の逆関数を求め、その定義域と値域を示し、グラフを同一座標平面上に図示しなさい。

解答と解説

元の関数 \(y = \sqrt{x+3}\) について:
定義域: \(x+3 \geq 0 \Rightarrow x \geq -3\)
値域: \(y \geq 0\)

逆関数を求める:
\(y = \sqrt{x+3}\)
両辺を2乗して（\(y \geq 0\) に注意）: \(y^2 = x+3\)
\(x\) について解く: \(x = y^2 - 3\)
\(x\) と \(y\) を入れ替える: \(y = x^2 - 3\)

逆関数 \(y = x^2 - 3\) の定義域と値域:
定義域: 元の関数の値域より \(x \geq 0\)
値域: 元の関数の定義域より \(y \geq -3\)

⭐ 答え: 逆関数は \(y = x^2 - 3\)。定義域は \(x \geq 0\)、値域は \(y \geq -3\)。

グラフの図示:

\(y = \sqrt{x+3}\) のグラフは、\(y=\sqrt{x}\) のグラフをx軸方向に-3平行移動したものです。(-3, 0) を通り、右上がりの曲線です。

\(y = x^2 - 3\)（ただし \(x \geq 0\)）のグラフは、放物線 \(y=x^2\) をy軸方向に-3平行移動したもののうち、\(x \geq 0\) の部分です。(0, -3) を頂点とし、y軸の右側を通る放物線の一部です。

これらのグラフは、直線 \(y=x\) に関して対称になります。

8. 理解度チェッククイズ ❓ ★★☆

  1. 関数 \(y = 3x - 6\) の逆関数は何ですか？
    1. \(y = \frac{1}{3}x + 2\)
    2. \(y = \frac{1}{3}x - 2\)
    3. \(y = -3x + 6\)
    解答と解説

    解答: a. \(y = \frac{1}{3}x + 2\)

    解説: \(y = 3x - 6\) を \(x\) について解くと、\(y+6 = 3x \Rightarrow x = \frac{y+6}{3} = \frac{1}{3}y + 2\)。
    \(x\) と \(y\) を入れ替えて \(y = \frac{1}{3}x + 2\)。

  2. 関数 \(y=f(x)\) とその逆関数 \(y=f^{-1}(x)\) のグラフは、どの直線に関して対称ですか？
    1. \(y = -x\)
    2. \(x\)軸
    3. \(y = x\)
    解答と解説

    解答: c. \(y = x\)

    解説: 逆関数のグラフは、元の関数のグラフを直線 \(y=x\) に関して対称移動したものです。

9. まとめと次回予告 🔖 ★★★

本日は逆関数について学習しました。重要なポイントは以下の通りです。

  • 逆関数は、元の関数の入力と出力を逆にした対応関係です。
  • 逆関数を求める手順は、「\(x\)について解く」→「\(x\)と\(y\)を入れ替える」です。
  • 逆関数では、元の関数の定義域が値域に、値域が定義域になります。この点を常に意識することが大切です。
  • 元の関数とその逆関数のグラフは、直線 \(y=x\) に関して対称です。

1. 関数の連続性の定義 📖 ★★★

連続性の直感的理解

関数の連続性とは、直感的にはグラフが途切れることなく滑らかに繋がっている状態を指します。

数学的な定義

関数 \(f(x)\) が \(x=a\) で連続であるとは、次の3つの条件がすべて満たされることをいいます。

  1. \(f(a)\) が定義されている（その点での関数の値が存在する）。
  2. \(\displaystyle \lim_{x \to a} f(x)\) が存在する（極限値が存在する）。
  3. \(\displaystyle \lim_{x \to a} f(x) = f(a)\) である（極限値と関数の値が一致する）。

極限値が存在するためには、左側極限（\(\lim_{x \to a-0} f(x)\)）と右側極限（\(\lim_{x \to a+0} f(x)\)）が一致する必要があります。

不連続とは

上記の3つの条件のうち、1つでも満たされない場合、関数 \(f(x)\) は \(x=a\) で不連続であるといいます。不連続にはいくつかのパターンがあります。

  • 極限値が存在しない（左側極限と右側極限が一致しない）。
  • 関数の値が定義されていない（グラフに穴が開いている）。
  • 極限値と関数の値が一致しない。

2. ガウス記号と関数の連続性 ✍️

ガウス記号の定義 ★★☆

ガウス記号 \([x]\) は、実数 \(x\) に対して、\(x\) を超えない最大の整数を表します。

例:

  • \([1.1] = 1\)
  • \([3] = 3\)
  • \([0.5] = 0\)
  • \([-1.5] = -2\) （-1.5を超えない最大の整数は-2です。-1ではありません）

例題: ガウス記号を含む関数のグラフと連続性 ★★★ ★★★

関数 \(f(x) = [x]\) のグラフを描き、\(x=1/2\) および \(x=1\) における連続性を調べなさい。

解答と解説

グラフの作成:

ガウス記号の定義に従って、いくつかの区間で \(f(x)\) の値を考えます。

  • \(0 \leq x < 1\) のとき、\([x] = 0\)
  • \(1 \leq x < 2\) のとき、\([x] = 1\)
  • \(2 \leq x < 3\) のとき、\([x] = 2\)
  • \(-1 \leq x < 0\) のとき、\([x] = -1\)

これをグラフにすると、階段状のグラフになります。

グラフは、各整数値のところでyの値がジャンプする形になります。各区間の左端は含み（黒丸）、右端は含まない（白丸）で表現します。

\(x=1/2\) における連続性:

  1. \(f(1/2) = [1/2] = 0\) （定義されている）
  2. \(\displaystyle \lim_{x \to 1/2-0} [x] = 0\) （\(x\)が0.5より少し小さいとき、例えば0.49なら[0.49]=0）
    \(\displaystyle \lim_{x \to 1/2+0} [x] = 0\) （\(x\)が0.5より少し大きいとき、例えば0.51なら[0.51]=0）
    よって、\(\displaystyle \lim_{x \to 1/2} [x] = 0\) （極限値が存在する）
  3. \(\displaystyle \lim_{x \to 1/2} [x] = 0\) であり、\(f(1/2) = 0\) なので、極限値と関数の値が一致する。

したがって、\(f(x) = [x]\) は \(x=1/2\) で連続です。

\(x=1\) における連続性:

  1. \(f(1) = [1] = 1\) （定義されている）
  2. \(\displaystyle \lim_{x \to 1-0} [x]\): \(x\) が1より少し小さいとき（例: 0.99）、\([x] = 0\)。よって左側極限は0。
    \(\displaystyle \lim_{x \to 1+0} [x]\): \(x\) が1より少し大きいとき（例: 1.01）、\([x] = 1\)。よって右側極限は1。
    左側極限(0)と右側極限(1)が一致しないため、\(\displaystyle \lim_{x \to 1} [x]\) は存在しません。

極限値が存在しないため、条件2が満たされません。したがって、\(f(x) = [x]\) は \(x=1\) で不連続です。

⭐ 答え:

  • \(x=1/2\) で連続
  • \(x=1\) で不連続

理解度チェッククイズ ❓ ★★☆

  1. 関数 \(f(x) = [x]\) は \(x=2.5\) で連続ですか、不連続ですか？
    1. 連続
    2. 不連続
    解答と解説

    解答: a. 連続

    解説: \(f(2.5) = [2.5] = 2\)。 \(\lim_{x \to 2.5-0} [x] = 2\)、\(\lim_{x \to 2.5+0} [x] = 2\) なので極限値は2。極限値と関数の値が一致するため連続です。

  2. ガウス記号 \([-3.14]\) の値は何ですか？
    1. \(-3\)
    2. \(-4\)
    3. \(-3.1\)
    解答と解説

    解答: b. \(-4\)

    解説: \(-3.14\) を超えない最大の整数は \(-4\) です。

⚠️ 間違いやすいポイント ★★★

  • ガウス記号の負の数の扱い: \([-2.3]\) のような負の数のガウス記号は、\(-2\) ではなく \(-3\) となります。「超えない最大の整数」という定義を正確に理解することが重要です。数直線をイメージすると分かりやすいです。
  • 連続性の条件の確認漏れ: 連続であるためには3つの条件すべてを満たす必要があります。極限値が存在しても、関数の値と一致しなければ不連続となります。
  • 極限値と関数の値の混同: \(\lim_{x \to a} f(x)\) は \(x\) が \(a\) に限りなく近づくときの \(f(x)\) の目標値であり、\(f(a)\) は \(x=a\) のときの実際の関数の値です。これらは必ずしも一致するとは限りません。
  • グラフの端点の扱い: ガウス記号のグラフなど、区間によって定義が変わる関数の場合、区間の端点を含むか含まないか（黒丸か白丸か）を正確に描くことが、連続性の判断に不可欠です。

✏️ 標準的な練習問題

問題 1 ★★☆ ★★☆

関数 \(f(x) = |x-2|\) について、\(x=2\) における連続性を調べなさい。

解答と解説

1. \(f(2) = |2-2| = |0| = 0\) （定義されている）

2. 左側極限: \(\lim_{x \to 2-0} |x-2|\)
\(x\) が2より少し小さいとき（例: \(x=1.9\)）、\(x-2\) は負の値をとるので、\(|x-2| = -(x-2) = -x+2\)。
よって、\(\lim_{x \to 2-0} (-x+2) = -2+2 = 0\)。

右側極限: \(\lim_{x \to 2+0} |x-2|\)
\(x\) が2より少し大きいとき（例: \(x=2.1\)）、\(x-2\) は正の値をとるので、\(|x-2| = x-2\)。
よって、\(\lim_{x \to 2+0} (x-2) = 2-2 = 0\)。

左側極限と右側極限が一致するので、\(\lim_{x \to 2} |x-2| = 0\) （極限値が存在する）。

3. \(\lim_{x \to 2} |x-2| = 0\) であり、\(f(2) = 0\) なので、極限値と関数の値が一致する。

したがって、\(f(x) = |x-2|\) は \(x=2\) で連続です。

⭐ 答え: \(x=2\) で連続

問題 2 ★★★ ★★★

関数 \( f(x) = \begin{cases} x+1 & (x \leq 1) \\ 3-x & (x > 1) \end{cases} \) について、\(x=1\) における連続性を調べなさい。

解答と解説

1. \(f(1)\) の値: \(x=1\) は \(x \leq 1\) の範囲に含まれるので、上の式 \(x+1\) を用います。
\(f(1) = 1+1 = 2\) （定義されている）

2. 左側極限: \(\lim_{x \to 1-0} f(x)\)
\(x\) が1より少し小さいので、\(x \leq 1\) の定義 \(f(x)=x+1\) を用います。
\(\lim_{x \to 1-0} (x+1) = 1+1 = 2\)。

右側極限: \(\lim_{x \to 1+0} f(x)\)
\(x\) が1より少し大きいので、\(x > 1\) の定義 \(f(x)=3-x\) を用います。
\(\lim_{x \to 1+0} (3-x) = 3-1 = 2\)。

左側極限と右側極限が一致するので、\(\lim_{x \to 1} f(x) = 2\) （極限値が存在する）。

3. \(\lim_{x \to 1} f(x) = 2\) であり、\(f(1) = 2\) なので、極限値と関数の値が一致する。

したがって、この関数は \(x=1\) で連続です。

⭐ 答え: \(x=1\) で連続

🔖 授業の要点まとめ ★★★

  • 関数の連続性: 関数 \(f(x)\) が \(x=a\) で連続であるとは、
    1. \(f(a)\) が定義されている。
    2. \(\lim_{x \to a} f(x)\) が存在する（左側極限 = 右側極限）。
    3. \(\lim_{x \to a} f(x) = f(a)\) が成り立つ。
    の3条件がすべて満たされること。
  • ガウス記号 \([x]\): 実数 \(x\) を超えない最大の整数。
    • 例: \([3.14] = 3\), \([-0.5] = -1\)。
    • 関数 \(y=[x]\) のグラフは階段状になり、各整数値で不連続（ジャンプ）となる。
  • 関数の連続性を調べる際は、定義された3つの条件を丁寧に確認することが重要。
